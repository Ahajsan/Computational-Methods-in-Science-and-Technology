# Computational Methods in Science and Technology 2024/2025

Repozytorium zawiera implementacje kluczowych algorytmów metod numerycznych oraz metod obliczeniowych nauki i techniki, realizowanych w ramach zajęć laboratoryjnych.


## Spis treści

1. **[Interpolation](./interpolation)**
   * Wyznaczanie wartości funkcji pomiędzy węzłami. Porównanie różnych technik: interpolacja wielomianowa Lagrange’a, różnice dzielone Newtona oraz funkcje sklejane (splajny).

2. **[Aproximation](./aproximation)**
   * Aproksymacja funkcji i zbiorów danych. W szczególności skupienie na aproksymacji trygonometrycznej oraz metodzie najmniejszych kwadratów.
   
3. **[Nonlinear Equations](./Nonlinear%20equations)**
   * Rozwiązywanie równań nieliniowych jednej zmiennej. Implementacja metod takich jak: metoda bisekcji, metoda Newtona-Raphsona oraz metoda siecznych.
   
4. **[Linear Equations](./linear%20equations)**
   * Rozwiązywanie układów równań liniowych. Metody bezpośrednie (eliminacja Gaussa) oraz metody it
