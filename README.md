# Geometric Algorithms 2024/2025

Repozytorium zawiera implementacje kluczowych algorytmów geometrii obliczeniowej przygotowanych w ramach zajęć laboratoryjnych.

## Zawartość

1. **[Line Orientation & Precision](./Line%20precision)**
   * Badanie orientacji punktów względem prostej oraz analiza precyzji obliczeń zmiennoprzecinkowych.
2. **[Convex Hull](./Convex%20Hull)**
   * Wyznaczanie otoczki wypukłej zbioru punktów algorytmami **Grahama** oraz **Jarvisa**.
3. **[Triangulation](./Triangulation)**
   * Algorytmy triangulacji wielokątów (m.in. podział na wielokąty monotoniczne).
4. **[Segment Intersections](./SweepLine/)**
   * Implementacja algorytmu zamiatania (**Bentley-Ottmann**) do wykrywania przecięć odcinków w czasie $O((n+k) \log n)$.
